== Description ==

又拍网(Yupoo)非官方的widget插件，可以显示来自又拍的照片。

目前支持以下特性：

*设置用户账号

*设置显示照片数量

*设置照片高度和宽度

== Installation ==

1.在Wordpress站点后台搜索Yupoo并安装激活

2.在Settings中设置用户账号等

3.在Widget中将此插件拖动到显示的位置
